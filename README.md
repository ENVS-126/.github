# Advanced Statistics for Environmental Science

## Instructor 

[An Bui](https://an-bui.com/)  

## Course description (from UCSB catalog)

Students will learn advanced concepts of probability and statistics, identify appropriate analyses to test hypotheses, and conduct all data analysis and visualization using current tools and practices within the field of environmental data science. Students will use environmental science topics to interpret and contextualize statistical findings. All work will rely on the R programming language, though no prior experience with R is required. Lower-division statistics and environmental studies will provide useful context for course material.
